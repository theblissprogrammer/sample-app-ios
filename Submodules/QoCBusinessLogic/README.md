## Installation

#### Carthage
You can use [Carthage](https://github.com/Carthage/Carthage) to install `MyModule` by adding it to your `Cartfile`:
```
github "iOSMentor/MyModule"
```

#### CocoaPods
Add `pod "MyModule"` to your `Podfile`.
